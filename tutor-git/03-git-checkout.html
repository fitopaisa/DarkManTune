Viajando en el tiempo

git checkout

podremos camviar a versiones atras . viajando en el teimpo
Necesitaremos copiar el numero del commit para viajar.

Ponermos:
git checkout 6d16bed17b02e51dc15fde57459beab7c8f3907a

Nos mandara de retorno a la version del commit que pongamos.

Para volver a la ultima version guaradada ponemos.
git checkout master .

Git reset

git reset --soft
borra los commit pero no borra el código. de tal manerea que peude dejar esa copia en el punto qeu queiras borrando las pequeñas actualizaciones que no te interesa guardar.
no boorra ni el codigo ni los archivos. Si borra los commit.
git reset --soft 1f1e4ccb15800375b289b45506b55f36c53175c1

git reset --hard
Borra los comit y el codigo
dejandote en la vesion de commit que elijas borrando tambien el codigo y los ficheros.
git reset --hard 1f1e4ccb15800375b289b45506b55f36c53175c1
