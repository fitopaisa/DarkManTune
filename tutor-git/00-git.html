Para instalar GIT

Lo primero pondremos tu nombre de usuario
git config --global user.name "DarkManTune"

Para ver qnuestro nombre poner lo mismo pero sin commillas
git config --global user.name

DarkManTune

Lo primero pondremos tu nombre de usuario
git config --global user.email "fitopaisa@hotmail.com"

Para ver nuestro imail poner lo mismo pero sin commillas
git config --global user.email

Para configurar los colores:
git config --global color.ui true

Para ver nuestra configuracion en GIT
git config --global --list
